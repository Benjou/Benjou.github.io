What this page is

This page is generated by an automated script that collects data from different river level sensors upstream of Oxford and South Hinksey and displays them in a way that makes sense and allows to appreciate the dynamics of floods.
This script is run every 2 hours in periods of flooding, and is run only seldomly in normal conditions. Check the date and time on the plots to see if they are up to date. If, for some reason, you want me to update the page, please send me an email.

What this page is not

This page is not any sort of accurate model to use to try and make predictions. The EA runs much better predictions. Do not base yourself on this for any sort of flood-related planning!
